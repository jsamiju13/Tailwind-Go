¿Que conforma una base de datos?

-Una Base de datos está conformada por múltiples elementos, entre ellos las Tablas, los Registros, Los campos, Las Claves y mucho más


Composición de la Base de datos

Tablas: Son la base de una base de datos relacional. Cada tabla está compuesta por filas y columnas. Las columnas representan los atributos de los datos (como nombre, edad, dirección), mientras que las filas contienen los registros individuales de esos dato.



Registros: Cada fila en una tabla se llama registro. Un registro es una colección de datos relacionados que se almacenan juntos. Por ejemplo, en una tabla de “Clientes”, cada fila podría representar un cliente diferente.

Campos: Las columnas de una tabla se llaman campos. Cada campo contiene un tipo específico de dato, como texto, número, fecha, etc. Por ejemplo, en una tabla de “Clientes”, podrías tener campos como “Nombre”, “Apellido”, "Correo Electrónico".



Claves Primarias: Una clave primaria es un campo o conjunto de campos cuyos valores son únicos en cada fila de una tabla. Sirve para identificar de forma unívoca cada registro en la tabla.



Claves Foráneas: Son campos que establecen una relación entre dos tablas. Una clave foránea en una tabla apunta a una clave primaria en otra tabla, permitiendo así la vinculación de datos entre tablas.



Claves candidatas: Una clave candidata es una superclave mínima. Esto significa que es un conjunto de columnas que puede identificar de manera única cada fila en una tabla, y no contiene ningún atributo redundante.
